# 更新日志

所有重要的更改都会记录在这个文件中。

## [1.1.0] - 2025-11-18

### 新增
- 🔐 **管理密码功能**
  - 支持非主机用户通过密码访问管理设置
  - 主机用户无需密码，自动识别
  - 管理会话有效期24小时
  - 支持退出管理会话
  - 配置文件中可设置管理密码

### 优化
- 📊 管理页面UI优化
  - 新增密码验证界面
  - 区分主机和管理员身份显示
  - 添加会话状态提示
- 🔒 安全性提升
  - Token基于会话管理
  - 自动清理过期会话
  - API请求统一鉴权

### 文档
- 📖 新增 `ADMIN_PASSWORD.md` 管理密码功能详细说明
- 📝 更新 `README.md` 配置说明
- 📋 更新 `CHANGELOG.md` 记录新功能

## [1.0.1] - 2025-11-06

### 新增
- 🔐 **二维码登录支持**（推荐使用）
  - 更安全，不需要保存密码
  - 不容易触发风控
  - 控制台显示二维码
  - 支持所有账号类型
- 📱 保留手机号密码登录方式（不推荐）

### 优化
- 📝 更新所有文档，添加二维码登录说明
- ⚙️ 简化配置文件结构
- 🔒 提高账号安全性

### 修复
- 🐛 修复 `NeteaseCloudMusicApi` 导入问题
- 🔧 修复搜索功能报错问题

## [1.0.0] - 2025-11-06

### 新增功能
- ✨ 初始版本发布
- 🎵 支持搜索网易云音乐歌曲
- 🎮 实现中心化播放模式
- 📱 响应式设计，支持移动端和桌面端
- 🔄 实时播放队列同步
- 🎨 美观现代的用户界面
- 🎧 HTML5 音频播放器
- 📋 播放队列管理（添加、删除、清空）
- ⏮️ ⏭️ 播放控制（上一首、下一首、播放/暂停）
- 📊 进度条和音量控制
- 🔌 WebSocket 实时通信
- 🔐 网易云账号登录支持

### 技术实现
- 使用 Vue 3 + Composition API
- 使用 Vite 作为构建工具
- 使用 Element Plus UI 组件库
- 使用 Express + Socket.io 实现后端
- 集成 NeteaseCloudMusicApi 开源项目
- 跨平台支持（Windows/Linux/Mac）

### 文档
- 📖 完整的 README.md
- 📘 详细的使用指南 GUIDE.md
- 📝 项目结构说明 PROJECT_STRUCTURE.md
- 🚀 一键启动脚本（Windows bat 文件）
